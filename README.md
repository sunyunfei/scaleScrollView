# scaleScrollView
一个滑动和缩放的图片展示demo，一句代码直接使用。
```
//obtain imageView index
    YFScrollView *scr = [[YFScrollView alloc]initWithDataArray:self.dataArray andImageIndex:indexPath.row andFrame:self.view.frame];
    [scr addInKeyWindow];
```
很简易的一个小demo，如果有需要的可以看看
